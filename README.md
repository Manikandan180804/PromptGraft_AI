## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- No installation required!

### Usage

1. Open `index.html` in your web browser
2. **Browse Templates** (Optional): Click "Browse Templates Library" to use pre-built prompts
3. Select your **field of work** from the dropdown (30+ options organized by category)
4. Choose your **optimization style**: Professional, Casual, or Technical
5. Enter your prompt in the text area (or use a template)
6. (Optional) Click "Advanced Options" to customize:
   - Include contextual information
   - Add constraints and requirements
   - Request examples in responses
   - Specify output format
7. Click **"Optimize Prompt"** to generate your improved prompt
8. Use the action buttons:
   - **Save**: Store the prompt locally for later use
   - **Export**: Download as JSON file
   - **Copy**: Copy to clipboard
   - **Regenerate**: Create a new optimization
9. Paste the optimized prompt into ChatGPT, Claude, Gemini, or any AI tool
